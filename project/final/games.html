Game Genres

Action Games

genre that emphasizes physical challenges, including hand-eye coordination and reaction time.

Some incredibly popular examples are Call of Duty and Battlefield.

Strategy Games

A strategy game has all of the following elements in varying degree: Players succeed (or lose) based on strategic decisions, not luck. Players have equal knowledge to play; no trivia. Play is based on multiple decisions a person could make on each turn with possible advantages and disadvantages each time.

Examples of some great strategy games include Civilization and Age of Empires.

Strategy game types: real time strategy or RTS is a game where you do not progress in turns instead everything is done in well, real-time.

Turn-based types: As the name states players play this type of gaming in turns, this type is where critical thinking is important.
